# chapelnik
A small web application that fetches Twitter stream for a couple of keywords

## Development

```bash
# install all dependencies
npm install
bower install
# build the frontend and watch for changes
gulp
# run the app in development environment
npm run dev
# run tests
npm run test
```
## Production

```bash
# install all dependencies
npm install
bower install
# build the frontend
npm run build
# run the app in development environment
npm run prod
```
